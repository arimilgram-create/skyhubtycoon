# Agent Instructions

This repository uses `AGENT.md` and `GAME_MEMORY.md` as the project handoff documents.

Before changing code or docs in this repository:

1. Read `AGENT.md`.
2. Read `GAME_MEMORY.md`.
3. Preserve the Unity 2022.3 LTS WebGL starter structure unless a task explicitly requires changing it.
4. Keep changes small and update `GAME_MEMORY.md` when project direction, completed features, bugs, next steps, or design decisions change.
